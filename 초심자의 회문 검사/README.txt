초심자의 회문 검사

C로 풀기 도전.
    - scanf와 %s로 문자열을 입력받을 예정.
    - for문으로 10번 반복을 하려함. -> 공백이라는 문제가 생길 것 같음.
    - 처음에 문자를 scanf와 %s로 입력을 받으면 얼마나 받았는지를 모를 수 있기에 for문으로 %c로 한 문자씩 받는걸로 변경.
    - 엔터키가 입력되면 i의 수가 늘어나지 않게 for문을 빠져나감. 중요한건 i는 엔터키를 포함하고 있다는 것. -> 다음 for문을 가기전에 i -= 1 해줌.
    - 처음에 count를 입력 받을때 숫자와 Enter키가 문자형으로 들어가버림 -> getchar로 해결
